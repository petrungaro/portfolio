About

Hi I'm Rob , a front-end developer based in Toronto and recent Juno grad.

I've got a constantly curious mind and an excitement for learning new technologies and building my skills. My work before coding left me with a strong appreciation for clean, modern, functional design. With web development, that has translated into writing code that provides the user with intuitive, informative content that is visually engaging.

When I'm not coding or learning, I enjoy doing a little woodworking, going for a swim, sinking into a video game, or biking around.
