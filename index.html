About

Hi I'm Rob , a front-end developer based in Toronto.

I've got a perpetually curious mind and an excitement for learning new technologies and building my skills. My work before coding left me with a strong appreciation for clean, modern, functional design. With web development, that has translated into writing code that provides the user with visually engaging, intuitive, and informative content that is inclusive and accessible.

Keep on scrolling to check out some of my recent projects.

When I'm not coding or learning, I enjoy doing a little woodworking, going for a swim, sinking into a video game, or biking around.
